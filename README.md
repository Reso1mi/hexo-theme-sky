## hexo-theme-sky:一款简洁的hexo博客主题

## 功能简介
hexo-theme-sky 是一款简洁轻量的 hexo 博客主题，主要包含以下功能：
- 语言切换，目前支持中文和英文
- 文章访问统计
- 文章目录
- 评论功能，目前使用的是gitment
- 代码高亮

## 安装
```
git clone https://github.com/iJinxin/hexo-theme-sky themes/even
npm install hexo-renderer-scss --save
```
修改配置文件中``` theme ```为 ```sky```
